# 🌐 简易海外网站导航

> "Across the Great Wall, we can reach every corner in the world"

一个轻量级的海外网站导航页面，收集了常用海外网站、学习资源链接，并集成机场订阅/VPN 入口。使用 **ToneCSS** 框架构建界面样式。

## ✨ 功能特点

- **常用海外站点分类展示** — 正经类、机长类、VPN 订阅、相关资料，分类清晰
- **响应式卡片布局** — 适配桌面与移动端，移动端自动切换为单列
- **轻量简洁** — 纯 HTML + CSS，无需 JavaScript，无需后端
- **ToneCSS 主题** — 基于 ToneCSS 框架，自定义配色与圆角

## 📂 项目结构

```
simple_index/
├── index.html          # 主页面
├── README.md           # 本文件
├── css/
│   ├── style.css       # 自定义样式
│   ├── tone.css        # ToneCSS 框架样式
│   ├── lib/            # ToneCSS 库文件
│   └── themes/         # ToneCSS 主题文件
└── proxys/
    └── edgetunnel.html # edgetunnel 代理页面
```

## 🚀 使用方式

直接用浏览器打开 `index.html` 即可，无需任何服务器环境。

## 🛠 构建说明

项目基于 [ToneCSS](https://tonecss.com/) 框架，自定义样式独立维护在 `css/style.css` 中。如需修改外观，编辑该文件即可。
